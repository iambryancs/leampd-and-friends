# leampd-and-friends
Your standard LAMP/LEMP stack and friends - Dockerized!

## Requirements
* docker
* docker-compose

## Getting started
1. Clone this repo inside your code's directory
2. Copy/rename sample-env to .env
3. Edit .env according to your needs
4. Run `docker-compose up`
